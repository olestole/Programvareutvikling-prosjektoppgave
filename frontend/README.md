# Skikkelig Fancy Hotell

> Front-end in Next.js

## Setup

```
$ git clone ...
$ cd frontend
$ yarn
```

## Development

```
$ yarn dev
# Open http://localhost:3000
```

## Linting and checking formatting

```
$ yarn lint
```

## Apply formatting

```
$ yarn prettier
```

## Run tests

```
$ yarn test
```
